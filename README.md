# Vrinda Store Data Analysis (Excel)

## 📊 Project Overview

This project analyzes the sales data of Vrinda Store using Microsoft Excel.
The goal of the analysis is to understand customer purchasing behavior, sales performance, and key business insights.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning

## 📁 Dataset

The dataset contains information about:

* Customer demographics
* Product categories
* Order details
* Sales amount
* Sales channels
* State-wise sales data

## 🔍 Key Insights

* Women customers contribute more to overall sales compared to men.
* The age group **26–35 years** generates the highest number of purchases.
* Top states contributing to sales are **Maharashtra, Karnataka, and Uttar Pradesh**.
* **Amazon, Flipkart, and Myntra** are the major sales channels.
* Most customers purchase products from **Clothing, Food, and Electronics** categories.
